# ModelerModRecipes
A mod to get all modded recipes from the game into Satisfactory Modeler.

# Information
This mod grabs all recipes and machines from your modded game and makes a specialized game_data, just for you. No more going in and manually writing your own game_data, it even gets icons for all those new items and machines! 

# Usage
- Get the path to your Satisfactory Modeler (Usually `C:\Program Files (x86)\Steam\steamapps\common\Satisfactory Modeler`) then run the command as follows:
- `/ModelerModRecipes "<path>"` or `/mmr "<path>"` (the quotes are important)
- Then you can open up Satisfactory Modeler normally and use your new recipes.

# Notice
- Remember to backup your `game_data.json` and `save.sfmd` just in case.
- For recipe multipliers, use the command on a save with the multiplier you want applied and use the default multiplier settings in Satisfactory Modeler, they are already accounted for.
